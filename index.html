Simulador de tienda de armas

Este simulador representa una tienda dentro de un videojuego donde el jugador puede comprar diferentes armas utilizando cuadros de diálogo de JavaScript. El usuario selecciona armas disponibles y el sistema calcula el total de oro gastado.

La interacción se realiza mediante prompt, confirm y alert, mientras que los resultados también se pueden observar en la consola de JavaScript.
